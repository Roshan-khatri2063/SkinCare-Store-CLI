# 💄 Skincare Store CLI (Python)

A terminal-based skincare store management system built using Python. This application allows users to interact with a simple store, manage products, and perform basic operations like purchasing items.

---

## 🚀 Features

* 🧴 Display available skincare products
* 🛒 Purchase products
* 📄 Read product data from file
* ✍️ Write/update product data
* 💰 Basic billing system
* 🖥️ Fully terminal-based interface

---

## 📂 Project Structure

```
skincare-store-cli/
│
├── main.py            # Entry point of the application
├── operations.py      # Core business logic
├── read_data.py       # Handles reading from products file
├── write_data.py      # Handles writing/updating data
├── products.txt       # Stores product information
└── README.md
```

---

## ⚙️ How It Works

* Product data is stored in `products.txt`
* `read_data.py` reads product details
* `write_data.py` updates product records
* `operations.py` controls program logic
* `main.py` runs the application
